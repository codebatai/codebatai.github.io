Privacy Policy & Terms of Service

ðŸ”’ Privacy Policy

Last updated: January 2025

1. Data Collection and Use

We commit to not collecting, not storing, not processing your plaintext data. The platform only handles end-to-end encrypted ciphertext with no decryption capability.

2. Information We Collect

  • Metadata: Task ID, timestamps, file size (encrypted), hash values
  • System Logs: System performance monitoring and error detection (no business data)
  • Evidence Packages: Event record hashes, Merkle trees, signatures (WORM storage)

3. Data Protection Measures

  • End-to-end encryption (AES-256-GCM)
  • microVM isolated execution
  • WORM evidence package storage
  • Access control and audit trails

4. Third-Party Services

We do not use third-party analytics or advertising services. On-premises deployments run entirely within your infrastructure with no external dependencies.

5. Your Rights

  • Query and export evidence packages at any time
  • Request deletion of account and related metadata
  • Use verification tools to independently audit evidence chains

For privacy-related questions, contact: privacy@codebat.ai
